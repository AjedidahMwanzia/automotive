SUSPENSIONS

Multi-Link Suspension

A multi-link suspension is a type of vehicle suspension with one or more longitudinal arms. A wider definition can consider any independent suspensions having three control links or more multi-link suspensions.

Rigid Axle Suspension

A beam axle, rigid axle or solid axle is a dependent suspension design in which a set of wheels is connected laterally by a single beam or shaft.

Macpherson Suspension

MacPherson strut is a type of automotive suspension system that uses the top of a telescopic damper as the upper steering pivot. It is widely used in the front suspension.

Double Wishbone Suspension

Double wishbone suspension allows each wheel to act and react independently from the others.

Independent Suspension

Independent suspension is any automobile suspension system that allows each wheel on the same axle to move vertically.

Air Suspension

An air suspension system is a style of vehicle suspension that's powered by an electric pump or compressor that pumps air into flexible bellows that are typically made out of a textile-reinforced type of rubber.
